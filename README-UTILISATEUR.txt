Instructions d'installation pour le Générateur de Podcast

1.  Décompressez le fichier .zip que vous avez téléchargé.

2.  Double-cliquez sur l'exécutable "Podcast Generator" pour lancer l'application.

3.  Au premier lancement, une fenêtre vous demandera de fournir votre clé API Google Gemini.

4.  Collez la clé que je vous ai fournie et cliquez sur OK.

5.  C'est tout ! L'application est prête à être utilisée pour les lancements futurs.
